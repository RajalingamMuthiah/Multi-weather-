Weather Dashboard Application

Project overview
Developed a full stack weather dashboard that allows users to register, log in, and track weather data for multiple cities. The application provides real time weather, forecast, history, and intelligent alerts using external APIs and database storage.

Key features

2.1 Secure user authentication using JWT
2.2 Personalized dashboard for each user
2.3 Add, delete, and mark cities as favorite
2.4 Real time weather data using OpenWeatherMap API
2.5 5 day weather forecast display
2.6 Weather history stored using MongoDB
2.7 Weather insights and alert recommendations
2.8 Clean and responsive frontend interface

Architecture

Frontend communicates with backend using REST APIs
Backend handles authentication, business logic, and API integration
MongoDB stores users, cities, and weather data
JWT is used to secure routes and sessions
